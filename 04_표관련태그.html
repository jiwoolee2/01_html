table 관련 태그

  1. table(표) 태그 
                    - 웹 문서에서 
                자료(data)를 정리하기 위해 
                사용하는 태그

                - table 태그는
                행, 열을 작성하기 위한　태그
                영역(판)을 지정하는 태그

  2. th(Table Header) 태그

                    - 열의 제목을 나타내는 태그

                - td와 같은 레벨로 작성(td 대신 th 작성 가능하다)

                - 굵은 글씨 + 가운데 정렬

  3. caption 태그

                    - 표 또는 이미지의 설명을 작성하는 태그

  4. thead 태그

                    - 테이블 상단 영역(컬럼 제목 영역)

  5. 태그

                    - 테이블 중단 영역(값 작성 영역)

  6. tfoot 태그

                    - 테이블 하단 영역(합계 영역)

  7. colspan(열 병합) 속성

    열로 나열된 방향(가로,옆) 병합

  8. rowspan(행 병합) 속성

    행으로 나열된 방향(세로,위/아래) 병합
