block 형식 요소

        - 화면(공간, 영역)을 수직 분할(가로로 자른걸 수직으로 분할했다고 함../층을 나눔)

        - width / height 크기 지정 속성 가능

        - block 형식 요소의 종류
        h1~h6(제목), p/pre(문단), hr(수평선), div(영역 분할), 시멘틱 태그(div의 이름을 변경한 것)

        - div 태그를 이용해서 화면을 쪼개는 역할(원하는 자리에 원하는 내용을 넣을 수 있음)

inline 형식 요소

        - 화면을 수평 분할
        (가로 화면/영역을 잘라서 열로 분할(김밥 썰기))

        - width / height 크기 속성 지정 불가능
        (작성된 내용 만큼의 크기만 차지함)

        - inline 형식 요소 종류
        b, i, strong, em, mark, span(영역 분할 담당)
inline 형식은 행을 잘라서 옆으로 만드는 성질을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다 (inline 형식 요소는 글자 취급)

block vs inline : 영역 지정 방식의 차이

        - block 형식 요소 : 항상 직사각형의 형태로 영역을 지정

        - inline 형식 요소 : 작성된 내용(content) 단위로 영역을 지정

inline-block 형식 요소

        - inline의 특징 중 열을 나눔
        - block의 특징 중 크기 지정

        두 특징만을 가져와 섞은 형식
        - inline-block 형식 요소 종류
        img 태그, iframe 태그

iframe 태그

        웹 문서(html) 내부에 또 다른 웹 문서(html)을 추가하는 태그
        (ex 네이버 카페-왼쪽,상단부분은 그대로고 안에 내용만 바뀜
        , 유튜브 동영상 첨부)
